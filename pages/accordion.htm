An accordion control groups related information within expandable and collapsable sections.

Accessible Accordion

Bootstrap Syntax

<div class="accAccordion"
data-src="path/file.xml"
data-defaultopen="#"
data-role="ContentPanelRoleName"
data-tabrole="TriggeringElementRoleName"
data-tabstate="TriggeringElementOpenState"
></div>

Attributes

  • class="accAccordion" : The bootstrap class name that specifies an accessible accordion.
  • data-src : The resource path.
  • data-defaultopen : The array index number of the accordion branch to open automatically when loaded. Leave blank to specify none.
  • data-role : The role name that is conveyed to screen reader users as beginning and ending boundary text for the expanded content. "Accordion" is set by default if no value is specified.
  • data-tabrole : The role name that is conveyed to screen reader users within the triggering link, which is appended to the visible link text. "Accordion" is set by default if no value is specified.
  • data-tabstate : The state name that is conveyed to screen reader users within the triggering link after activation, which is appended to the data-tabrole value. "Open" is set by default if no value is specified.

XML Syntax

<?xml version="1.0" encoding="UTF-8" ?>
<group>
<link name="Link Text Name" class="linkClassName" ><![CDATA[
HTML markup for accordion content here.
]]></link>
</group>
